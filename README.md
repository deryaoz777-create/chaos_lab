# chaos_lab

Bu klasörde küçük denemeler var. Şu an için en “ürün gibi” çalışan parça:

## Horary (Eğlence Modu)

Soru astrolojisini çok hafifçe oyunlaştırıp “o anın ruhu” üzerinden kısa bir çıktı üretir. Ciddî astrolojik danışmanlık değildir.

### Kurulum

```bash
pip install -r requirements.txt
```

### Çalıştırma

```bash
python horary_app.py
```
